# Barbearia Alura

Este projeto foi uma tarefa do curso de Front-End da Alura, consiste na criação de um site de uma barbearia.

## Tecnologias Utilizadas
* HTML
* CSS

## Conteúdos abordados
* Introdução de Iframe
* Display flex e Grid
* Tabelas
* Formulários
